Minecraft Java Edition 1.21

名称: 棘巧试炼

版本: Java

发布日期: 2024年6月13日

新内容

方块

方块名称 介绍
雕纹铜块 铜块的变种，有着X形花纹。
铜门 会自然生成于试炼密室。

物品

  • 刷怪蛋 - 加入了沼骸刷怪蛋和旋风人刷怪蛋。
  • 试炼钥匙 - 可从试炼密室的箱子、饰纹陶罐和试炼刷怪笼战利品中获得。

生物

  • 沼骸 - 骷髅的新变种，拥有不同的生命值和攻击间隔。
  • 旋风人 - 类似烈焰人的幽灵状敌对生物，自然通过试炼刷怪笼生成于试炼密室中。

非生物实体

  • 风弹 - 由旋风人或风弹物品发射的弹射物。
